Cloud Search Service (CSS)
==========================

The CSS client is the command-line interface (CLI) for
the Cloud Search Service (CSS) API and its extensions.

For help on a specific `css` command, enter:

.. code-block:: console

   $ openstack css help SUBCOMMAND
